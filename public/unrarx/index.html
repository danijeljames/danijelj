What’s New!

Universal Binary version of UnRarX is now available. Version 2.2 includes current universal binary versions of unrar and par2 command line applications. UnRarX now includes automatic update functionality thanks to Sparkle Framework by Andy Matuschak. Many thanks to Andy! You can download UnRarX in zip format by clicking here.

UnRarX 2.2 source will be available soon.

Brief Description:

UnRarX is a Mac OS X Cocoa application that allows you to expand rar archives and restore corrupted or missing archives using par2.
